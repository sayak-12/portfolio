Featured Project #1

Ourkennel

A comprehensive Pet Care Website built with a modern UI, featuring a login system and a dedicated social media platform for pet owners to connect, share, and celebrate their pets. The website includes features like pet adoption and buying and a street pet welfare section for reporting and assisting stray animals. With a MongoDB backend, cloud-based image storage, and responsive design, it ensures seamless performance and accessibility. Additional features include a personalized dashboard for managing pets, community forums, and curated resources for pet care.

  • MongoDB

  • Node.js

Featured Project #2

Greenhack

GreenHack Agrotech WebApp is a cutting-edge platform designed to revolutionize urban agriculture and gardening. It features a modern UI and integrates IoT sensors with machine learning models to provide real-time insights into air temperature, soil moisture, and NPK levels. The platform supports smart water management with automated DC motor control for water pumps, preventing waterlogging and ensuring optimal crop health. It offers features like crop recommendations, field monitoring, and detailed analytics. GreenHack is a one stop solution for sustainable and efficient farming practices.

  • NoSQL
  • WebSocket
  • Flask
  • React
  • Node

Featured Project #3

Tech.care

Tech.Care is a robust hospital management system built on the MERN stack with Next.js for server-side rendering and optimized performance. It eases hospital operations with features like patient registration and records management, appointment scheduling, doctor patient communication, and billing automation. The system includes a secure MongoDB backend for data management, real time notifications, and a responsive design for seamless access on any device. With cloud integration for reports and analytics, Tech.Care ensures efficient healthcare administration, making it an ideal solution for modern hospitals.

  • MongoDB
  • TailwindCSS
  • React
  • Node
